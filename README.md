# こうかとんのアクション

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 横スクロールアクション，障害物をジャンプでよけながら進むゲーム
* 主人公キャラクターこうかとん，敵キャラクタートマト

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキーでジャンプすることができる
* こうかとんがトマトにふれる，または奈落に落ちるなどでゲームオーバーとなる
* 旗に振れたらゴール

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 横スクロール
* ゲームオーバー判定
* クリア条件

### 分担追加機能
* 残機（担当：髙橋）：主人公キャラクターのライフ．0になるまでゲームを繰り返すことができる機能．主人公キャラクターが奈落判定，及び，敵判定と衝突した際，残機表示のされたミス画面へと遷移．リスタート時と同様，ミスするたびにステージはリセットされる．
* 敵キャラクターの動き（担当：根本）：敵キャラクターのランダム生成．主人公キャラクター，及び，敵キャラクター画像の差し込み.
* アイテムの実装（担当：武内）：5秒の間，敵キャラクターにぶつかっても敵判定と衝突しない無敵状態の効果を付与できる機能．
* スコアの実装（担当：鈴木）：主人公キャラクターが敵キャラクターを倒したときにポイントが加算される機能．
* トラップの実装（担当：鈴木）：初見殺しの罠．
* サウンドの実装（担当：鈴木）：サウンドの追加.
