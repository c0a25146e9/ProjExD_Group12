# こうかとんのアクション

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要

* 横スクロールアクション，障害物をジャンプでよけながら進むゲーム
* 主人公キャラクターこうかとん，敵キャラクタートマト

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキーでジャンプすることができる
* こうかとんがトマトにふれる，または奈落に落ちるなどでゲームオーバーとなる
* 旗に振れたらゴール

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 横スクロール
* ゲームオーバー判定
* クリア条件

### 分担追加機能
* 残機：主人公キャラクターのライフ．0になるまでゲームを繰り返すことができる機能．
* 敵キャラクターの動き：ランダム生成、敵キャラを倒すと100点スコア加算
* アイテムの実装：無敵になる，一回的にぶつかっても耐えられる，敵を倒せるビームを打てるようになる，など様々な効果を付与できる機能．
* スコアの実装：主人公キャラクターが敵キャラクターを倒したとき，コインを入手したときに加算される機能．
