# こうかとんのアクション

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要

* 横スクロールアクション，障害物をジャンプでよけながら進むゲーム
* 主人公キャラクターこうかとん，敵キャラクタートマト

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキーでジャンプすることができる
* こうかとんがトマトにふれる，または奈落に落ちるなどでゲームオーバーとなる
* 旗に振れたらゴール

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 横スクロール
* ゲームオーバー判定
* クリア条件

### 分担追加機能
* 残機（担当：髙橋）：主人公キャラクターのライフ．0になるまでゲームを繰り返すことができる機能．主人公キャラクターが奈落判定，及び，敵判定と衝突した際，残機表示のされたミス画面へと遷移．リスタート時と同様，ミスするたびにステージはリセットされる．
* 敵キャラクターの動き（担当：根本）：同じ動きをする個体．（追尾する個体も？）主人公キャラクターの体力やライフを減らしたり，倒されたときスコアとして加算されたりする機能．
* アイテムの実装（担当：武内）：無敵になる，1度敵キャラクターにぶつかっても耐えられる，敵を倒せるビームを打てるようになる，など様々な効果を付与できる機能．
* スコアの実装（担当：鈴木）：主人公キャラクターが敵キャラクターを倒したとき，コインを入手したときに加算される機能．
