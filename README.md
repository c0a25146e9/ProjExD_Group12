# こうかとんのアクション

![image alt](photo/play_screen.png)

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 横スクロールアクション，障害物をジャンプでよけながら進むゲーム
* 主人公キャラクターこうかとん，敵キャラクタートマト

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキーでジャンプすることができる
* こうかとんがトマトにふれる，または奈落に落ちるなどでゲームオーバーとなる
* こうかとんがトマトを倒すと10ポイントゲット
* 青いアイテムを取ると少しの間無敵状態になる
* 旗に振れたらゴール

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 横スクロール
* ゲームオーバー判定
* クリア条件

### 分担追加機能
* 残機（担当：髙橋）：主人公キャラクターのライフ．主人公キャラクターが奈落判定，及び，敵判定と衝突した際，残機表示のされたミス画面へと遷移．リスタート時と同様，ミスするたびにステージはリセットされる．
* 敵キャラクター（担当：根本）：敵キャラクターのランダム生成．主人公キャラクター，及び，敵キャラクター画像の差し込み.
* アイテム（担当：武内）：5秒間，敵キャラクターにぶつかっても敵判定と衝突しない無敵状態の効果を付与できる機能．
* 制限時間（担当：武内）：制限時間の表示．制限時間を迎えると強制的にミス画面へと遷移．
* スコア（担当：鈴木）：主人公キャラクターが敵キャラクターを倒したときにポイントが加算される機能．
* トラップ（担当：鈴木）：初見殺しの罠．
* サウンド（担当：鈴木）：敵判定と衝突したとき，奈落に落ちたとき，クリアしたときに流れるサウンドの追加．
